# Vikas Yadav — Academic Research Portfolio

🌐 **Live:** [vikasy95.github.io](https://vikasy95.github.io)

A modern, interactive academic portfolio website showcasing research publications, experience, and expertise in LLMs, agentic reasoning, and multilingual AI.

## Features

- 🎨 **Beautiful design** — Clean, modern UI with smooth animations
- 📄 **Interactive paper cards** — 20 most recent papers with abstract SVG diagrams
- 🔍 **Category filtering** — Filter papers by research area
- 📱 **Fully responsive** — Works great on mobile, tablet, and desktop
- ⚡ **Zero dependencies** — Pure HTML, CSS, and vanilla JS
- 🔎 **SEO optimized** — Schema.org structured data included

## Customization

### Adding your profile photo
Drop your photo as `images/profile.jpg` and the placeholder will be replaced automatically.

### Updating papers
Edit the `PAPERS` array in `js/main.js` to add or modify publications.

## Tech Stack
- HTML5 + CSS3 (custom properties, grid, animations)
- Vanilla JavaScript (IntersectionObserver, SVG generation)
- Google Fonts (Inter)
